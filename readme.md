# solar
[![built with nix](https://builtwithnix.org/badge.svg)](https://builtwithnix.org)

Create beautiful generative solar system images using
[gruvbox](https://github.com/morhetz/gruvbox-contrib) as a color palette.

If you have [`nix` flakes
install](https://nixos.wiki/wiki/Flakes#Installing_flakes)ed, all you have to do
to get a random solar system image is:
```
nix run github:ihsanturk/solar output.png
```
> If you don't know what [`nix`](https://nixos.org) is, you probably missing
> out a great software.
